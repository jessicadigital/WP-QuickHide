# WP-QuickHide
A small WordPress plugin to add a quick hide button to all pages. A button is added to the lower right of all non-admin pages for browser windows >720px wide that temporarily shows the Google search prompt. This can be used by users to hide sensitive content on their screen.

## How to use
The plugin is contained within the `quickhide` folder. Upload the whole folder to your `/wp-content/plugins/` directory, and click `Activate` within the Plugins screen. Alternatively, zip the `quickhide` folder and upload this through the `Add New` button at the top of the Plugins page.

This plugin was created for a specific project. It is provided as-is: if you discover any problems, please raise an Issue.
